
Demka & APIDemkaForVue - Интернет-магазин обуви
📋 Описание проекта
Данный репозиторий содержит два приложения, реализующие один и тот же функционал интернет-магазина обуви:

Demka - десктопное приложение на WPF (C#)

APIDemkaForVue - веб-приложение на Vue.js + ASP.NET Core Web API

Оба приложения предоставляют полноценный функционал для управления каталогом обуви с разграничением прав доступа (администратор/пользователь/гость).

🛠 Технологический стек
Backend (APIDemkaForVue)
ASP.NET Core Web API

Entity Framework Core

MS SQL Server

REST API архитектура

Frontend (Vue.js)

Vue Router для навигации

Desktop (Demka)
WPF (Windows Presentation Foundation)

Entity Framework Core

MS SQL Server

✨ Функциональность
Общая для обоих приложений:
🔐 Авторизация и роли
Роль	Возможности
Гость	Просмотр каталога
Пользователь	Просмотр каталога + поиск
Администратор	Полный доступ (CRUD товаров и заказов)
👟 Каталог товаров
Отображение товаров в виде карточек

Изображения товаров (300x200 px)

Информация о товаре: артикул, описание, цена, скидка, количество, производитель, поставщик

Цветовая индикация скидок (>15% - зеленый фон)

Визуальное выделение товаров с нулевым остатком

Бейдж "Новинка" для товаров, добавленных за последние 7 дней

🔍 Фильтрация и сортировка
Поиск по названию и описанию

Фильтр по категориям (Женская/Мужская обувь)

Фильтр по поставщикам (Kari/Обувь для вас)

Сортировка по цене (возрастание/убывание)

Сортировка по количеству (возрастание/убывание)

Сортировка по дате добавления (новые/старые)

Отображение количества найденных товаров

Сброс всех фильтров одной кнопкой

📦 Управление товарами (только для администратора)
Добавление нового товара

Редактирование существующего товара

Удаление товара (с проверкой на наличие в заказах)

Загрузка изображений с проверкой размера (300x200)

Валидация полей:

Артикул: максимум 6 символов

Описание: максимум 500 символов

Скидка: не более 99%

Количество: должно быть положительным

📊 Управление заказами (только для администратора)
Просмотр всех заказов

Статистика заказов (всего/новые/завершенные)

Фильтрация по статусу и дате

Поиск по коду заказа

Редактирование только новых заказов

Завершенные заказы доступны только для просмотра

Валидация дат: от сегодня до +1 год

Цветовая индикация статуса заказа
